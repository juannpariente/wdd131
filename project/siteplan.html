PC Essentials

This name reflects that the site is about the essential components of a PC.

Site Purpose

The purpose of this site is to inform users about essential PC components and help them explore hardware by simulating a virtual shopping experience. The site will provide detailed descriptions, images, and prices for each component, allowing users to learn and interact at the same time.

Scenarios

What components do I need to build a basic gaming PC?

How much would it cost to build a PC with the parts I choose?

Color Schema

Navy Blue (#001f3f): Used as the background color for headings.

Black (#000000): Used as the background color for the components section.

Green (#28a745): Used for buttons

Black (#000000) and White (#FFFFFF): Used for typography depending on the background color for readability.

Typography

Headings: Inter, bold (700)

Body text: Inter, regular (400)

Buttons and labels: Inter, semibold (600)
